## 概览

智能设计智能体的核心使命：用户只需输入极简信息（我是谁、要什么、在哪里、做什么），AI自动完成所有深度思考并生成世界Top3水准的设计提示词。

### 核心价值

- **极简输入**：用户只需说清楚4个问题（我是谁、要什么、在哪里、做什么）
- **深度思考**：AI自动完成品牌洞察、用户画像、理念、文化、战略、美学、意境的完整构建
- **顶级输出**：AI自动生成世界Top3水准的设计提示词
- **情感触发**：AI自动触发仰望、渴求、安心、信任等情感反应
- **行动驱动**：AI自动驱动用户采取行动（如主动报名）

## 核心架构

## 输入解析层

### 核心任务

从用户极简输入中提取所有必要信息，构建完整的设计语境。

### 4个核心问题解析

#### 1. "我是谁" - 品牌身份解析

**提取维度**：
- **品牌类型**：个人品牌/企业品牌/活动品牌/项目品牌
- **行业领域**：心理学/教育/科技/艺术/文化等
- **品牌层级**：顶级/高端/大众/入门
- **品牌调性**：专业/温暖/创新/传统等
- **品牌资产**：现有Logo、色彩、字体、视觉语言

**解析算法**：
```
输入："周六在广州新华书店6楼开一场50人的心理学沙龙"
↓
解析我是谁：
- 品牌类型：活动品牌
- 行业领域：心理学
- 品牌层级：专业级（心理学暗示专业性）
- 品牌调性：专业、温暖、洞察力、深度
- 品牌资产：需补充（假设无现有资产）
```

#### 2. "要什么" - 目标解析

**提取维度**：
- **设计类型**：海报/宣传册/网页/社交媒体/包装
- **核心目标**：品牌曝光/用户获取/销售转化/情感连接
- **情感目标**：仰望/渴求/安心/信任/行动
- **业务目标**：50人报名/品牌认知/用户留存
- **视觉风格**：具体风格（如果用户提供）

**解析算法**：
```
输入："海报设计"
↓
解析要什么：
- 设计类型：海报
- 核心目标：用户获取（50人报名）
- 情感目标：信任、安心、渴求
- 业务目标：50人报名
- 视觉风格：需AI推断（基于心理学沙龙）
```

#### 3. "在哪里" - 场景解析

**提取维度**：
- **物理空间**：书店/美术馆/商场/办公室/户外
- **空间属性**：文化空间/商业空间/公共空间/私密空间
- **空间氛围**：安静/热闹/正式/休闲
- **空间符号**：书籍/艺术/商业/自然
- **空间定位**：6楼（暗示需要上下楼，需要引导）

**解析算法**：
```
输入："广州新华书店6楼"
↓
解析在哪里：
- 物理空间：书店
- 空间属性：文化空间
- 空间氛围：安静、知识、思考
- 空间符号：书籍、知识、智慧
- 空间定位：6楼（需要引导、需要吸引人流上楼）
```

#### 4. "做什么" - 活动内容解析

**提取维度**：
- **活动类型**：沙龙/讲座/展览/演出/派对
- **活动规模**：50人（小型、私密、深度）
- **活动主题**：心理学（洞察力、情感、成长、治愈）
- **活动价值**：知识获取/情感连接/社交互动/个人成长
- **活动目标**：报名参与

**解析算法**：
```
输入："开一场50人的心理学沙龙"
↓
解析做什么：
- 活动类型：沙龙（深度交流、互动）
- 活动规模：50人（小型、私密、深度）
- 活动主题：心理学（洞察力、情感、成长、治愈）
- 活动价值：知识获取、情感连接、个人成长
- 活动目标：报名参与
```

### 语境构建

**构建维度**：
- **时间语境**：周六（周末、休闲、学习时间）
- **地点语境**：新华书店（文化、知识、安静）
- **活动语境**：心理学沙龙（深度、私密、成长）
- **用户语境**：50人（小型、精选、高质量）
- **情感语境**：需要信任、安心、渴求

**构建算法**：
```
语境 = 时间语境 + 地点语境 + 活动语境 + 用户语境 + 情感语境
      = 周末 + 文化空间 + 深度沙龙 + 小型群体 + 信任渴求
      = 一个在周末文化空间的小型深度沙龙，需要建立信任并激发参与渴求
```
