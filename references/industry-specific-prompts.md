## 行业专属提示词生成规则

### 核心规则

当用户输入设计需求时，AI自动执行以下步骤：

1. **行业识别**：从用户输入中识别行业类型
2. **行业匹配**：匹配到6大行业分类之一（认知/审美/情感/功能/体验/社交）
3. **深度洞察加载**：加载该行业的深度洞察
4. **提示词生成**：基于深度洞察生成完整提示词
5. **情感触发验证**：验证情感触发是否有效
6. **输出优化**：优化提示词确保顶级输出

### 完整提示词结构

## 认知需求类提示词模板

### 知识教育类专属模板

#### 精神内核

**核心精神**：深度、专业、成长、连接
**哲学思考**：知识的价值在于改变认知，认知的改变在于付诸实践
**诗意语言**：如月光般柔和的蓝色，如烛光般温暖的橙色，如书籍般质感的白，知识如光，照亮前路；成长如翼，助力飞翔
**意境描述**：宁静而有力量，深度而有温度，专业而温暖

#### 色彩策略

```
- 主色调：蓝色系（知识、专业、信任）
  - 具体色值：#2B5B9D（深蓝）、#4A90E2（中蓝）、#87CEEB（浅蓝）
  - 占比：60%-70%
- 辅助色：橙色系（温暖、成长、活力）
  - 具体色值：#FF8C00（深橙）、#FFA500（橙）、#FFD700（金）
  - 占比：20%-30%
- 点缀色：金色系（价值、卓越、权威）
  - 具体色值：#FFD700（金）、#FFA500（金橙）
  - 占比：10%-15%
```

#### 情感触发策略

**仰望感触发**：
```
- 权威性：顶级导师的智慧光芒，权威机构的深度底蕴，专业知识的卓越品质
- 稀缺性：50人精选席位，限定时间专属机会，深度洞察的独到视角
- 卓越性：前沿知识的前瞻性，深度洞察的独特性，实用技能的高价值
```

**渴求感触发**：
```
- 可能性：技能提升的无限可能，认知突破的全新视角，职业发展的跃迁机会
- 梦想：成为行业专家，实现人生理想，改变职业轨迹
- 渴望：解决当前瓶颈，满足成长需求，实现职业突破
```

**安心感触发**：
```
- 信任：权威机构的专业背书，顶级导师的可靠指导，成熟体系的稳定支持
- 安全：知识学习的安全感，成长过程的安全保障，职业发展的安全路径
- 可靠：稳定的持续支持，可预期的成长结果，可靠的后续服务
```

**信任感触发**：
```
- 专业：顶级导师的专业背景，权威机构的资质认证，深度知识的专业质量
- 真诚：导师的真诚分享，机构的真诚承诺，服务的真诚态度
- 承诺：明确的技能提升承诺，具体的职业发展保证，可靠的成长效果兑现
```

**行动触发**：
```
- 紧迫感：50人席位仅剩X个，限时优惠即将结束，深度洞察机会难得
- 价值感：高价值知识高回报成长，低时间成本高职业收益，一次性投入终身受益
- 社交压力：同行业精英都在学习，朋友圈成功人士都在参加，同事朋友都在推荐
```

#### 技术参数

```
- 负空间占比：40%-50%
- 字号比例：72pt : 36pt : 24pt : 16pt = 4.5 : 2.25 : 1.5 : 1
- 对比度：≥ 75%
- 质感：书籍质感、知识质感、温暖质感
- 字体：Helvetica、Roboto（现代、专业、易读）
- 色彩空间：RGB/CMYK
- 分辨率：300dpi
```
