## 第一部分：美学底层逻辑

### 1.1 认知神经科学基础

#### 视觉加工机制

**早期视觉处理（V1-V4区域）**：
- **边缘检测**：V1区域检测亮度梯度（明暗边界）
  - 应用：创造明确的边缘与轮廓
  - 提示词："sharp edges, clear contours, high contrast boundaries"
- **方向选择性**：V1/V2区域检测线条方向（水平/垂直/对角）
  - 应用：主导方向引导视觉流动
  - 提示词："dominant diagonal lines, directional flow 45°"
- **颜色对立**：V4区域处理色相对比（红-绿、蓝-黄）
  - 应用：利用色相对比创造视觉冲击
  - 提示词："complementary color contrast, red-green opposition"

**中期视觉处理（IT区域）**：
- **形状识别**：物体形状与轮廓整合
  - 应用：简化形状，提升识别速度
  - 提示词："simple geometric shapes, recognizable silhouettes"
- **纹理分析**：表面纹理与材质识别
  - 应用：高质量纹理增强真实感
  - 提示词："high-resolution texture, material fidelity"
- **色彩整合**：色彩情感与联想
  - 应用：色彩心理学精准应用
  - 提示词："color psychology applied, emotional resonance"

**高级视觉处理（前额叶皮层）**：
- **意义建构**：视觉元素的意义解读
  - 应用：符号与隐喻的精准使用
  - 提示词："symbolic elements, metaphorical meaning"
- **情感反应**：视觉刺激的情感触发
  - 应用：情感化设计元素
  - 提示词："emotional triggers, affective design"
- **记忆编码**：视觉元素的记忆强化
  - 应用：记忆锚点与重复曝光
  - 提示词："memory anchors, repeated exposure"

#### 注意力机制

**自上而下注意力**：
- **目标导向**：根据任务需求分配注意力
  - 应用：明确视觉目标与焦点
  - 提示词："clear visual goal, focused attention"
- **预期驱动**：基于预期的视觉搜索
  - 应用：利用视觉习惯与预期
  - 提示词："expectation-based design, visual habits"

**自下而上注意力**：
- **显著性驱动**：高对比度、鲜艳色彩吸引注意力
  - 应用：创造视觉显著性
  - 提示词："visual salience, high contrast, vibrant colors"
- **新奇性驱动**：新颖元素吸引注意力
  - 应用：适度新颖性打破习惯化
  - 提示词："novelty, break from convention, unexpected"

**注意力容量限制**：
- **7±2原则**：同时注意的元素限制
  - 应用：信息分组，降低认知负荷
  - 提示词："information grouping, chunking, reduce cognitive load"
- **认知隧道**：深度关注时的周边忽略
  - 应用：设计视觉隧道引导深度关注
  - 提示词："visual tunnel, deep focus, peripheral fade"

#### 记忆机制

**工作记忆**：
- **容量限制**：保持信息的数量限制（3-5个元素）
  - 应用：简化信息，符合工作记忆容量
  - 提示词："simplified information, working memory compatible"
- **持续激活**：信息的保持时间（15-30秒）
  - 应用：创造持续激活的视觉元素
  - 提示词："sustained activation, visual reinforcement"

**长时记忆**：
- **语义记忆**：知识与概念的存储
  - 应用：利用文化符号与共同记忆
  - 提示词："cultural symbols, shared knowledge"
- **情景记忆**：个人经验的存储
  - 应用：创造个人化的视觉体验
  - 提示词："personalized experience, narrative design"
- **程序记忆**：技能与习惯的存储
  - 应用：利用视觉习惯与直觉
  - 提示词："visual habits, intuitive design"

**记忆强化**：
- **重复曝光**：重复增强记忆
  - 应用：适度重复，避免疲劳
  - 提示词："moderate repetition, avoid fatigue"
- **情感强化**：情感事件记忆更深刻
  - 应用：情感化记忆锚点
  - 提示词："emotional memory anchors, affective events"
- **独特性强化**：独特事件记忆更深刻
  - 应用：创造独特视觉记忆点
  - 提示词："unique visual memory points, distinctive elements"

### 1.2 格式塔心理学深度应用

#### 完形原则深度解析

**1. 接近性原则**
- **原理**：距离近的元素被感知为一组
- **深度应用**：
  - 信息分组间距 < 元素间距 × 0.5
  - 视觉流线通过接近性引导
  - 层级关系通过间距差异表达
- **提示词模板**：
  ```
  格式塔接近性应用：
  - Group related elements: [元素A, 元素B, 元素C] with spacing < 50% of element size
  - Visual flow: Create directional flow through proximity
  - Hierarchy: Express hierarchy through spacing differences (primary: 20px, secondary: 40px)
  ```

**2. 相似性原则**
- **原理**：相似的元素被感知为一组
- **深度应用**：
  - 颜色相似性：主色调统领全图
  - 形状相似性：几何形状的一致性
  - 纹理相似性：纹理模式的重复
- **提示词模板**：
  ```
  格式塔相似性应用：
  - Color similarity: Dominant color [主色] with 70% coverage
  - Shape similarity: Consistent geometric shapes [形状类型]
  - Texture similarity: Repeating texture pattern [纹理描述]
  ```

**3. 闭合性原则**
- **原理**：人脑倾向于闭合不完整形状
- **深度应用**：
  - 不完整形状创造视觉参与感
  - 暗示性轮廓激发想象
  - 负空间形成暗示形状
- **提示词模板**：
  ```
  格式塔闭合性应用：
  - Incomplete shapes: Create [形状类型] with 70% closure
  - Suggestive contours: Implied shapes through negative space
  - Negative space formation: Suggest [暗示形状] through arrangement
  ```

**4. 连续性原则**
- **原理**：人眼倾向于跟随连续路径
- **深度应用**：
  - 视觉路径的连续性设计
  - 元素的对齐创造连续流
  - 线条的延伸引导视线
- **提示词模板**：
  ```
  格式塔连续性应用：
  - Visual path: Create continuous flow from [起点] to [终点]
  - Alignment: Align elements to create continuous lines
  - Line extension: Extend lines to guide eye movement
  ```

**5. 图底关系**
- **原理**：图形与背景的相互关系
- **深度应用**：
  - 可逆的图底关系创造深度
  - 模糊的图底边界增加趣味
  - 动态的图底转换创造活力
- **提示词模板**：
  ```
  格式塔图底关系应用：
  - Reversible figure-ground: Create ambiguity between figure and ground
  - Blurred boundaries: Soft transition between [图形] and [背景]
  - Dynamic figure-ground: Allow figure-ground interchange
  ```

**6. 共同命运**
- **原理**：同方向运动的元素被感知为一组
- **深度应用**：
  - 动态元素的运动一致性
  - 视觉方向的一致性
  - 趋势线的一致性
- **提示词模板**：
  ```
  格式塔共同命运应用：
  - Motion consistency: Elements move in [方向] with [速度]
  - Visual direction: Consistent directional elements
  - Trend lines: Aligned trend elements
  ```

**7. 简洁性原则**
- **原理**：人脑倾向于感知最简单的形式
- **深度应用**：
  - 简化复杂形状
  - 消除不必要元素
  - 寻找本质结构
- **提示词模板**：
  ```
  格式塔简洁性应用：
  - Simplify complex shapes: Reduce [复杂形状] to [简单形状]
  - Eliminate unnecessary: Remove [不必要元素]
  - Find essence: Capture core structure of [对象]
  ```

### 1.3 符号学深度应用

#### 符号学三要素

**能指**：
- **定义**：符号的物质形式
- **类型**：
  - 图像符号：相似性关系（照片、绘画）
  - 指示符号：因果性关系（烟→火）
  - 象征符号：约定性关系（十字→基督教）
- **应用**：
  - 选择恰当的能指形式
  - 能指的视觉设计质量
  - 能指的多义性控制
- **提示词模板**：
  ```
  符号学能指应用：
  - Signifier type: [图像符号/指示符号/象征符号]
  - Visual quality: [能指视觉质量描述]
  - Ambiguity control: [多义性控制策略]
  ```

**所指**：
- **定义**：符号所指代的概念
- **类型**：
  - 外延意义：字面意义
  - 内涵意义：联想意义
  - 意识形态意义：社会文化意义
- **应用**：
  - 明确的核心意义
  - 丰富的联想意义
  - 深刻的社会文化意义
- **提示词模板**：
  ```
  符号学所指应用：
  - Denotation: [字面意义]
  - Connotation: [联想意义列表]
  - Ideological meaning: [社会文化意义]
  ```

**符号链**：
- **定义**：符号之间的关联链条
- **类型**：
  - 转喻：基于邻近性（王冠→国王）
  - 隐喻：基于相似性（时间就是金钱）
  - 提喻：基于部分与整体（帆→船）
- **应用**：
  - 构建清晰的符号链
  - 符号链的层次结构
  - 符号链的文化适配
- **提示词模板**：
  ```
  符号学符号链应用：
  - Sign chain: [符号A] → [符号B] → [符号C]
  - Hierarchy: Primary: [主要符号], Secondary: [次要符号], Tertiary: [第三符号]
  - Cultural adaptation: [文化适配策略]
  ```

### 1.4 现象学深度应用

#### 具身认知

**身体-心智连接**：
- **原理**：认知根植于身体经验
- **应用**：
  - 身体运动引导视觉（视觉跟随身体动作）
  - 触觉体验影响视觉感知
  - 空间感知基于身体图式
- **提示词模板**：
  ```
  现象学具身认知应用：
  - Body movement: Visual elements follow [身体运动模式]
  - Tactile influence: Visual texture suggests [触觉体验]
  - Spatial schema: Space organized based on [身体图式]
  ```

#### 氛围营造

**现象学氛围**：
- **原理**：氛围是人与环境的整体体验
- **应用**：
  - 光、色、质、声的综合营造
  - 空间的情绪感染力
  - 时间的感知（流动、静止）
- **提示词模板**：
  ```
  现象学氛围营造应用：
  - Light: [光照描述] creating [情绪效果]
  - Color: [色彩描述] evoking [情感反应]
  - Texture: [质感描述] suggesting [触觉体验]
  - Space: [空间描述] creating [氛围效果]
  - Time: [时间感知描述] (static/flowing/eternal)
  ```

## 第二部分：设计底层逻辑

### 2.1 功能-形式-情感三角模型

#### 功能维度

**核心问题**：设计是否解决问题？

**功能分析框架**：
1. **明确功能目标**：
   - 主要功能：设计的核心目的
   - 次要功能：辅助功能
   - 潜在功能：未来扩展功能

2. **功能实现机制**：
   - 功能路径：从输入到输出的路径
   - 交互流程：用户与设计的交互过程
   - 反馈机制：设计的反馈方式

3. **功能评估标准**：
   - 有效性：功能是否实现
   - 效率性：实现功能的速度
   - 可靠性：功能的稳定性

**提示词模板**：
```
功能维度设计：
Primary function: [主要功能描述]
Secondary functions: [次要功能列表]
Functional path: [功能路径描述]
Interaction flow: [交互流程描述]
Feedback mechanism: [反馈机制描述]
Evaluation: Effectiveness [1-10], Efficiency [1-10], Reliability [1-10]
```

#### 形式维度

**核心问题**：设计是否具有美感？

**形式分析框架**：
1. **视觉形式要素**：
   - 色彩：色相、饱和度、亮度
   - 形状：几何形态、有机形态
   - 质感：纹理、材质、光泽
   - 空间：正负空间、深度、层次

2. **形式组织原则**：
   - 平衡：对称、非对称、放射
   - 对比：大小、色彩、质感
   - 韵律：重复、变化、渐进
   - 统一：一致性、连贯性

3. **形式评估标准**：
   - 美感：视觉愉悦度
   - 协调性：形式要素的和谐
   - 独特性：形式的创新性

**提示词模板**：
```
形式维度设计：
Visual elements:
- Color: [色相] [饱和度%] [亮度%]
- Shape: [形状类型], [几何/有机]
- Texture: [质感描述], [材质类型]
- Space: [正负空间比例], [深度描述]

Formal principles:
- Balance: [平衡类型], [平衡描述]
- Contrast: [对比类型], [对比描述]
- Rhythm: [韵律类型], [韵律描述]
- Unity: [统一性描述]

Evaluation: Aesthetics [1-10], Harmony [1-10], Uniqueness [1-10]
```

#### 情感维度

**核心问题**：设计是否引发情感？

**情感分析框架**：
1. **情感类型**：
   - 基本情感：喜、怒、哀、惧、惊、厌
   - 复杂情感：希望、羞愧、自豪、感激
   - 品牌情感：信任、忠诚、归属、自豪

2. **情感触发机制**：
   - 色彩情感：色彩引发的情感反应
   - 形状情感：形状引发的情感反应
   - 质感情感：质感引发的情感反应
   - 叙事情感：故事引发的情感反应

3. **情感评估标准**：
   - 情感强度：情感的强烈程度
   - 情感准确性：情感与品牌的一致性
   - 情感持久性：情感的持续影响

**提示词模板**：
```
情感维度设计：
Target emotions:
- Primary emotion: [主要情感], intensity [1-10]
- Secondary emotions: [次要情感列表], intensity [1-10]
- Brand emotions: [品牌情感列表], intensity [1-10]

Emotional triggers:
- Color: [色彩] triggers [情感]
- Shape: [形状] triggers [情感]
- Texture: [质感] triggers [情感]
- Narrative: [故事] triggers [情感]

Evaluation: Intensity [1-10], Accuracy [1-10], Persistence [1-10]
```

## 第三部分：排版底层逻辑

### 3.1 信息架构

#### 层级系统

**层级定义**：
- **一级层级**（L1）：最重要信息（标题、核心信息）
- **二级层级**（L2）：次重要信息（副标题、重要信息）
- **三级层级**（L3）：辅助信息（正文、详细说明）
- **四级层级**（L4）：补充信息（注释、标签）

**层级视觉表达**：
1. **大小层级**：
   - L1 : L2 : L3 : L4 = 1.618 : 1 : 0.618 : 0.382 (黄金比例)
   - 字号示例：72pt : 45pt : 28pt : 17pt

2. **位置层级**：
   - L1：左上角或中心（最高视觉权重）
   - L2：左上角或中心附近（高视觉权重）
   - L3：左侧或底部（中视觉权重）
   - L4：底部或边缘（低视觉权重）

3. **色彩层级**：
   - L1：最高对比度
   - L2：高对比度
   - L3：中对比度
   - L4：低对比度

**提示词模板**：
```
信息架构设计：
Hierarchy system:
- Level 1: [L1内容], size [字号], position [位置], contrast [对比度]
- Level 2: [L2内容], size [字号], position [位置], contrast [对比度]
- Level 3: [L3内容], size [字号], position [位置], contrast [对比度]
- Level 4: [L4内容], size [字号], position [位置], contrast [对比度]

Size ratio: L1:L2:L3:L4 = 1.618:1:0.618:0.382
Visual weight: L1 [最高], L2 [高], L3 [中], L4 [低]
```

#### 视觉流线

**流线类型**：
1. **F型流线**（西文阅读习惯）：
   - 起点：左上角
   - 路径：左上→右→左下→右
   - 应用：传统排版、长文本

2. **Z型流线**（注意力引导）：
   - 起点：左上角
   - 路径：左上→右上→左下→右下
   - 应用：广告、海报

3. **螺旋流线**（动态引导）：
   - 起点：任意位置
   - 路径：螺旋式向中心或外扩散
   - 应用：创意排版、艺术作品

4. **自由流线**（有机引导）：
   - 起点：自然起始点
   - 路径：有机流动
   - 应用：艺术排版、手写风格

**提示词模板**：
```
视觉流线设计：
Flow pattern: [F型/Z型/螺旋/自由]
Start point: [起点位置]
Path: [路径描述]
Element placement:
- [元素A]: Position [位置], Flow sequence [顺序]
- [元素B]: Position [位置], Flow sequence [顺序]
- [元素C]: Position [位置], Flow sequence [顺序]
```

#### 信息组块

**组块原理**：
- **7±2原则**：工作记忆容量限制
- **组块大小**：3-7个元素为一个组块
- **组块层次**：组块可以嵌套

**组块视觉表达**：
1. **空间组块**：
   - 通过间距分组：组块内间距 < 组块间间距
   - 通过容器分组：使用边界框或背景
   - 通过对齐分组：左对齐、居中对齐

2. **视觉组块**：
   - 通过色彩分组：相同颜色的元素
   - 通过形状分组：相同形状的元素
   - 通过大小分组：相似大小的元素

3. **语义组块**：
   - 通过意义分组：相关信息的集合
   - 通过功能分组：相同功能的元素
   - 通过类型分组：相同类型的元素

### 3.2 视觉韵律

#### 韵律类型

**重复韵律**：
- **定义**：相同元素的重复
- **应用**：创造一致性、强化记忆
- **提示词**："repeating elements, consistent rhythm, memory reinforcement"

**交替韵律**：
- **定义**：两个或多个元素的交替
- **应用**：创造活力、避免单调
- **提示词**："alternating elements, dynamic rhythm, avoid monotony"

**渐进韵律**：
- **定义**：元素的渐进变化（大小、色彩、位置）
- **应用**：创造流动感、引导视线
- **提示词**："progressive change, flowing rhythm, guide eye movement"

**随机韵律**：
- **定义**：元素的随机分布
- **应用**：创造自然感、打破规则
- **提示词**："random distribution, natural feel, break from rules"

#### 韵律参数

**重复频率**：
- **高频重复**：元素密集重复
  - 应用：创造节奏感、强化视觉冲击
  - 提示词："high frequency repetition, rhythmic density, visual impact"

- **中频重复**：元素适度重复
  - 应用：创造平衡感、维持视觉兴趣
  - 提示词："medium frequency repetition, balanced rhythm, sustained interest"

- **低频重复**：元素稀疏重复
  - 应用：创造简洁感、强调重点
  - 提示词："low frequency repetition, minimalist rhythm, emphasize focus"

**变化幅度**：
- **小幅度变化**：细微变化
  - 应用：创造微妙感、精致感
  - 提示词："subtle variation, refined detail, sophisticated"

- **中幅度变化**：适度变化
  - 应用：创造活力感、动态感
  - 提示词："moderate variation, dynamic energy, visual interest"

- **大幅度变化**：强烈变化
  - 应用：创造冲击感、戏剧性
  - 提示词："dramatic variation, strong impact, theatrical"

**提示词模板**：
```
视觉韵律设计：
Rhythm type: [重复/交替/渐进/随机]
Repetition frequency: [高频/中频/低频]
Variation amplitude: [小幅度/中幅度/大幅度]

Rhythm elements:
- Element A: [描述], repetition [次数], variation [幅度]
- Element B: [描述], repetition [次数], variation [幅度]
- Element C: [描述], repetition [次数], variation [幅度]

Rhythm effect: [韵律效果描述]
```
